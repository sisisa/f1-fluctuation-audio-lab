# f1-fluctuation-audio-lab

自然界に存在する心地よいリズム「1/fゆらぎ（ピンクノイズ）」をデジタル音声処理に応用し、音声に癒やしの質感を付与する実験的DSP（デジタル信号処理）プロジェクトです。
主にポッドキャストなどの長尺コンテンツでの利用を想定しています。

## プロジェクトの目的と技術的な課題解決

単に「声の背景でノイズを流す」だけでは、声とノイズが分離してしまい、自然なゆらぎを感じられません。また、声全体の音量を単純に揺らすだけでは、機械的な震え声になってしまいます。
本プロジェクトでは、この課題に対して以下の論理的なアプローチで解決を図っています。

1. 周波数帯域ごとの自然なゆらぎの適用
   声を周波数成分（スペクトル）に分解し、「低周波の揺れは強く、高周波の揺れは弱い」という1/fゆらぎの厳密な特性を、声の成分ごとに直接掛け合わせています。
2. 位相の不規則さによる「温かみ」の再現
   合成音声特有の完全に規則的な波の形（位相）に対して、1/f特性を持った微細な乱れを付与することで、人間の声帯特有の自然な息遣いやかすれを表現しています。

## 技術的な工夫と設計思想

長く安全に使い続けられる実用的なツールを目指し、以下の設計を意識しています。

* 「役割分担」によるディレクトリ構成
  ノイズ生成、音声処理、ファイル操作の責務をモジュールとして明確に分割し、保守性を高めています。
* 型安全性とエラーの防止
  Pythonの型ヒント（Type Hints）を導入し、各機能の入出力を明確にすることで、堅牢な設計を行っています。
* 制作現場を想定したバッチ処理の自動化
  JSONフォーマットの注文書（台本データ）を読み込み、複数の音声を全自動で連続生成する仕組みを構築しました。

## プロジェクト構成

```text
f1-fluctuation-audio-lab/
├── src/
│   ├── contents/             # 生成したい音声の台本（JSON）を入れるフォルダ
│   │   └── sample_episode.json
│   └── f1_fluctuation/
│       ├── generator.py      # 1/fノイズ生成の専用機能
│       ├── processor.py      # 音声の周波数・位相を処理する専用機能
│       └── utils.py          # ファイル入出力の便利ツール群
├── examples/
│   ├── generate_batch.py     # フォルダ内の台本を全自動で音声化するスクリプト
│   └── generate_from_text.py # 単一のテキストから生成する基本スクリプト
│   └── generate_from_mic.py # 人の音声から生成する基本スクリプト(まだ未着手)
├── requirements.txt          # 必要なライブラリの一覧
└── README.md                 # 本ドキュメント

## 使い方
### 1. 準備
必要なライブラリをインストールします。
```bash
pip install -r requirements.txt
```

### 2. 全自動バッチ処理
`src/contents/` フォルダの中に、以下のような形式で台本（JSONファイル）を配置します。

```json
{
  "output_name": "episode_1.wav",
  "alpha": 0.15,
  "text": "おはようございます。自然のゆらぎとともに、心穏やかな一日を。"
}
```

以下のコマンドを実行すると、フォルダ内のすべてのJSONファイルを読み込み、自動でWAVファイルを生成します。

```Bash
python examples/generate_batch.py
```
